# Vaccine
Permite ejecutar una serie de inyecciones SQL sobre una URL pasada por argumento al programa.

# Caracteristicas
- Detecta 2 tipos diferentes de bases de datos

- Diferentes pruebas en los metodos GET y POST: 
    - union
    - error
    - boolean
    - time
    - blind
    - out-of-band
    - stacked

- Cuando haya un positivo se obtiene la siguiente información: 
    - parametros vulnerables
    - payload usado
    - nombres de bdd
    - tablas de bdd
    - nombre de las columnas
    - base de datos completa.
